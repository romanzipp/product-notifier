# Nike Sneaker Availability Scraper

## Build

```
go build main.go
./main
```

## Configuration

- Supports [Pushover](https://pushover.net/#apps) notifications
- Sending native OS notifications

```
NIKE_URL=https://www.nike.com/de/t/air-force-1-luxe-herrenschuh-86CTL1
SIZES=44,44.5

PUSHOVER_APP_TOKEN=
PUSHOVER_USER_TOKEN=
```
